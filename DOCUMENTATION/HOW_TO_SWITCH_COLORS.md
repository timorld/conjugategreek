## ğŸ”„ How to Switch Palettes

### Method 1: Copy & Paste (Easy!)

1. Open `style.css`
2. Find the section "Greek Heritage Color Palette" (around line 295)
3. Open `DEVELOPMENT_SCRIPTS/colors-playful-rainbow.css`
4. Copy all the content
5. Replace the color section in `style.css`
6. Save and refresh your browser

### Method 2: Quick Replace

Replace this line in `style.css`:
```css

```

With:
```css

```

Then replace all the gradient colors below it with the ones from `colors-playful-rainbow.css`
